A simple telegram bot used to plan our evenings.

### Dependencies

- Python 3.8+
- git
- [Poetry](https://python-poetry.org/)

### Installation

Clone the repo:

```
git clone https://github.com/ODKclan/odkbot.git
```

Run:

```console
cd odkbot
poetry install
```

Create a file called `credentials.json`:

```json
{
  "token": "your-token-goes-here"
}
```

### Launch the bot

Run:

```console
poetry run odkbot
```
